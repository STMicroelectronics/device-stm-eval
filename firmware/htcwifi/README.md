# firmware htcwifi #

## Containing ##
* `htc_9271.fw`: firmware required for TP-Link HTC TL WN722.

## Dependencies ##

Linux kernel config : CONFIG_WLAN_VENDOR_ATH=y CONFIG_ATH9K_HTC=m CONFIG_ATH9K=m
Required kernel modules : ath.ko ath9k_hw.ko ath9k_common.ko ath9k.ko ath9k_htc.ko
